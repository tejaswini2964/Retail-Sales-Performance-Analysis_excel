# 📊 Retail Sales Performance Analysis using Microsoft Excel

## 📌 Project Overview

This project analyzes the sales performance of a retail company using Microsoft Excel. It demonstrates practical data analysis techniques including data lookup, logical functions, aggregation functions, Pivot Tables, and Conditional Formatting to generate meaningful business insights.

This project was completed as part of a Data Analytics training program to strengthen Excel skills used in real-world business reporting.

---

## 🎯 Objectives

- Analyze retail sales data using Microsoft Excel.
- Retrieve product details using lookup functions.
- Categorize sales performance based on business rules.
- Calculate employee bonuses.
- Generate summary reports.
- Create Pivot Tables for business insights.
- Apply Conditional Formatting for better visualization.

---

## 📂 Dataset

The workbook contains the following worksheets:

| Sheet | Description |
|--------|-------------|
| Product_Master | Product details including Product ID, Product Name, Category and Price |
| Sales_Data | Retail sales transactions |
| Summary | Aggregate calculations using Excel formulas |
| Pivot_Region | Total Sales by Region |
| Pivot_Category | Total Sales by Category |
| Pivot_Salesperson | Salesperson-wise Sales Analysis |

## 📋 Tasks Performed

### ✔ Product Lookup
- Retrieved Product Name using VLOOKUP.
- Retrieved Product Price using VLOOKUP.

### ✔ Sales Classification
Created performance categories:
- High
- Medium
- Low

### ✔ Region Classification
Mapped regions into business zones using IFS.

### ✔ Bonus Calculation
Calculated bonus using Nested IF.

### ✔ Sales Analysis
- Total Sales by Category
- Total Sales by Region
- Quantity Sold by Salesperson

### ✔ Order Analysis
- Furniture Orders Count
- Multi-condition Order Count

### ✔ Pivot Reports
Created Pivot Tables for:
- Region-wise Sales
- Category-wise Sales
- Salesperson-wise Sales

### ✔ Conditional Formatting
- High Sales Highlight
- Low Sales Highlight
- Top 3 Sales
- Data Bars
